# 🌌 Coriolis RPG - Tradução PT-BR (Foundry VTT)

Tradução do sistema **Coriolis: The Third Horizon** para o Português Brasileiro no Foundry VTT. Esta é uma tradução feita de fã para fã, não oficial e sem fins lucrativos, e pode conter erros.

## 📦 O que já está traduzido (v1.0.0)

Contém nessa versão:
* Ficha (Junto a alterações no CSS pra comportar os textos traduzidos)
* Aflições
* Poderes de pássaro (Garuda)
* Itens de combate
* Talentos
* Manobras de tripulação
* Equipamento
* Veículos
* Upgrades de veículo

---

## ⚙️ Como Instalar

Para instalar esta tradução diretamente no seu Foundry VTT, siga os passos abaixo:

1. Abra o Foundry VTT e vá até a aba **Add-on Modules**.
2. Clique no botão **Install Module**.
3. No campo **Manifest URL** (na parte inferior da janela), cole o link abaixo:
`https://raw.githubusercontent.com/Isaias-Alves/coriolis-rpg-traducao-pt-br/refs/heads/main/module.json`
4. Clique em **Install**.
5. Abra o seu mundo e ative este módulo junto com o **Babele**.
6. Coloque a linguagem preferida nas configurações de linguagem do mundo para **Português Brasil**

---

## ⚠️ Requisitos

Para que a tradução funcione perfeitamente, você precisará ter instalado e ativado no seu mundo:
* O sistema oficial de **Coriolis**.
* O módulo de tradução **Português Brasil** da interface do Foundry
* O módulo **Babele** (que é o motor responsável por aplicar os textos traduzidos nas fichas).
